# Image Process

本專案收錄多種影像處理與分析相關的程式實作，涵蓋基礎影像運算、濾波、銳化、去雜訊、幾何變換、邊緣偵測等主題。  
與一般使用現成影像函式庫的方式不同，本專案的重點在於**從零實作每個影像處理步驟**，  
透過數學公式與矩陣運算**完整理解影像處理的原理與運作機制**。

---

## 專案理念
本專案旨在以最基礎的程式邏輯實現影像處理流程，而非直接呼叫高階函式庫。  
透過自行撰寫函式，明確掌握：
- 每一步影像轉換的數學意涵  
- 濾波與卷積的實際運算過程  
- 影像特徵在像素層級的變化  

這樣的方式能更深入理解影像處理背後的理論。
